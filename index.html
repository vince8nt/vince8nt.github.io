Connected Component Finder

C

Depth First Search

A program that finds the connected components an unweighted, directed graph. Connected components are maximized, cyclic sub-graphs. The program takes in two arguments, an input file and an output file. The input file lists the starting vertex along with the graph represented as an adjacency list. Depth first search is used to find the sub-graphs.
