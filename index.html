Depth First Search

A program that finds the connected components an unweighted, directed graph. Connected components are maximized, cyclic sub-graphs. Tic Tac Toe

Python

Player vs computer connect-4. Computer's AI uses Depth First Search with depth 8. When the 8th move is reached without a guaranteed win or loss, a non-recursive position rating method is used. This method uses common connect-4 strategy to get a rough estimate of who the position favors, and by how much. With this, the computer will determine which move guarantees the best position 8 moves in the future.
